# Ninja Converter
- This converter allows the user to upload PDF/PPTX file and generate into images.

## Requirements
- Python 3.6+
- Virtualenv 15.1.0+
- Pip 20.2.3+
- Docker 19.03.6+
- Docker-compose 1.17.1+
- Npm 7.6.0+
- Node 15.11.0+
- Sass 1.34.0

## Setup
```bash
$ cd client
$ npm start
```
